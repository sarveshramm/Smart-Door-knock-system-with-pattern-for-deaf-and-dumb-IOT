##  Project Description  
**KnockSense** is an assistive IoT project designed to help **deaf and mute individuals** detect door knocks using **vibration feedback, LCD messages, and a buzzer alert**.  
It classifies different knock counts into meaningful alerts, ensuring **safety, accessibility, and inclusiveness**.

---

## ✅ Features  
- ✅ Detects knock patterns:  
  - **3 knocks → Husband**  
  - **4 knocks → Son**  
  - **5 knocks → Warning / Unknown Person**  
- ✅ Vibrates using a **tactile vibration motor**  
- ✅ Displays visitor info on a **16×2 I2C LCD**  
- ✅ Buzzer feedback for additional alert  
- ✅ WiFi-ready ESP8266 for **future mobile alerts**  
- ✅ Low-cost and beginner-friendly design

---

##  Hardware Used  
- **NodeMCU ESP-12E / ESP8266**  
- **Vibration Sensor (DO type)**  
- **Vibration Motor**  
- **16×2 I2C LCD (0x27)**  
- **Buzzer**  
- **Jumper Wires**  
- **Breadboard / PCB**

---

##  Project Overview  
This system gives **multi-sensory alerts**—LCD text, buzzer, and vibration—so users who cannot hear knocks can still identify visitors.  
Perfect for **deaf, mute, and elderly individuals**.
